VS Code 插件安装

本指南详细介绍如何安装 AgentLog VS Code 插件。

环境准备​

  • VS Code: ^1.96.0 或更高版本
  • Node.js: 22.x（推荐使用 nvm 管理 Node 版本）
  • Git

安装步骤​

1. 安装插件​

在 VS Code 中搜索 AgentLog，点击安装。

或者 直接下载 VSCode 插件

2. 开始使用​

安装完成后，VS Code 侧边栏会出现 AgentLog 面板，表示安装成功。

插件会自动启动后台服务（端口 7892）。

配置​

在 VS Code 设置（settings.json）中配置 AgentLog。

配置项默认值说明
agentlog.backendUrlhttp://localhost:7892后台服务地址
agentlog.autoCapturetrue是否自动捕获 AI 交互
agentlog.captureReasoningtrue是否捕获推理过程
agentlog.autoBindOnCommittruecommit 时自动绑定最近未绑定的会话
agentlog.debugfalse开启调试日志

其他安装方式​

如果你不使用 VS Code，可通过 Docker 部署后端服务：

  • Docker 部署 - 支持任意 MCP 客户端（Cline、Cursor、OpenCode、Roo Code 等）
