今天我们发布了 **AgentLog v1.1.2**，这是继 v1.1.0 Phase 1 之后的又一次重要更新，核心主题是 **OpenClaw 生态深度集成**。

<!--truncate-->

## 🎯 核心更新

### 1. openclaw-agentlog Skill（自动存证）

这是本次更新的主角！通过 OpenClaw Skill 机制，AgentLog 可以**自动为你的 AI Agent 会话存证**：

- **自动 Hooks**：session_start、before_tool_call、after_tool_call、agent_end、llm_output 全链路捕获
- **Reasoning 过程保存**：支持 DeepSeek-R1 等推理模型的思考链
- **Tool Call 记录**：包含完整 args 和 result，不再只有工具名
- **Trace Handoff**：任务交接时自动传递 Trace 上下文
- **Git Commit Binding**：会话与代码版本精确关联

安装方式：
```bash
clawhub install agentlog
```

### 2. agentlog-daily-report Skill（日报自动化）

打工人的福音！让 AI Agent 自动生成工作汇报：

- 从会话数据自动提取工作内容
- 支持按日期范围查询
- 生成结构化日报，一键导出

### 3. 后端增强

- `/api/traces` 新增 `parentTraceId` 字段，支持 Trace 链式关联，方便追踪多 Agent 协作场景

## 🔧 其他改进

- CI 流程优化：移除已废弃的 darwin-x64 构建（macos-13 runner）
- 文档完善：新增设计文档、审计文档、E2E 测试文档
