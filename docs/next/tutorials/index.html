使用教程

本文档提供 AgentLog 的详细操作参考。

侧边栏面板​

打开 VS Code 左侧边栏的 AgentLog 图标：

区域说明
未提交的 AI 会话当前工作区的游离会话
历史 Commit 记录已绑定 Commit 的会话，按时间倒序

常用操作​

查看会话详情​

点击任意会话节点，在右侧 Webview 查看完整交互记录，支持 Markdown 渲染和推理过程折叠。

手动绑定会话​

  1. 在侧边栏右键点击游离会话
  2. 选择「绑定到最新 Commit」或「选择 Commit 绑定」
  3. 选择目标 Commit 完成绑定

导出数据​

  1. 点击侧边栏顶部的「导出」按钮
  2. 选择格式：中文周报、PR 说明、JSONL、CSV
  3. 配置时间范围和筛选条件
  4. 点击「生成」保存文件

复活上下文​

  1. 在侧边栏找到历史 Commit 节点并展开
  2. 找到目标 AI 会话，点击「查看详情」
  3. 在详情页点击「复活上下文」按钮
  4. 内容自动复制到剪贴板，粘贴到 AI Agent 聊天框

工作区筛选​

侧边栏顶部提供筛选器：

  • 按仓库路径过滤会话
  • 按模型类型过滤
  • 按时间范围过滤

最佳实践​

  • 完成逻辑完整的任务后及时提交并绑定
  • 避免长时间累积大量游离会话
  • 重要项目定期导出备份
